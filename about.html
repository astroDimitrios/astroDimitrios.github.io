About

My name is Dimitrios and I'm currently a science teacher (physics specialist) in a secondary school in Nottingham, UK.

I first started coding when I was in secondary school and was formally taught Python and C++ at university (can you guess which I prefer?). I decided to start making these resources partly out of necessity for the BTEC Unit 16 module Astronomy and Space. My other motivation is my interest in visualisation. You'll notice that a lot of the activities have supplementary python files that make more complicated plots that are easier on the eye and easier to extract data from.

I also wanted to create a place where I could store astronomy related data files for educational use and found it hard finding data in a suitable format online.

When I'm not coding I'll probably be baking or taking photos which you can see here!
